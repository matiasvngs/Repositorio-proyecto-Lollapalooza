# Documentación

° ¿Cómo se realizó el proceso de visualización?

1. Primeramente me fije en los comentarios de la entrega anterior, donde solucione el primer problema es hacer coordinar los datos, arreglando varias faltas ortográficas y corrigiendo mayúsculas para conseguir una armonía en cada estadística. 
2. El segundo paso que tome fue revisar nuevamente mis datos, ya que desde ahí formaría mi gráfico. Desde mi base de datos sobre las ediciones de cada Lollapalooza pude sacar que la mayor diferencia se veía en los minutos que se le entregaban por género en cada edición, así que pude tener una vista general de lo que quería plantear.
3. El siguiente paso fue la búsqueda del gráfico que mejor le quedara a mi idea, esta búsqueda fue hecha en la página de Vega-Altair, así revisando ejemplos y encontrado mi selección que era el gráfico de barras apiladas.
4. Desde ahí descargué la base de datos y la librería de Altair, la dos ya incluidas en mi Google colab, tuve gran dificultad con varias confusiones en el camino, lo que me llevo a apoyarme en IA y en videos tutoriales de como hacer gráficos, hasta que llegue a un código que me tomara lo que buscaba mostrar.
5. Ahí pude activar el código y ver la primera visualización sobre mi gráfico, donde había un gran problema, me salían más categorías de las que yo tenía, diferenciándose por términos de Mayúscula y minúscula, entonces mi siguiente paso fue arreglar la base de datos para coordinar solo las tres opciones (Femenino, Masculino y Mixto).
6. Desde ahí, volví a incluir mi base de datos dentro de Google Colab y activando el código se pudo visualizar como quería mi gráfico de barras apiladas, con una imagen que destaca la cantidad de minutos que tienen los hombres en cada edición en comparación a las mujeres.
7. Finalmente, sin tener problemas con los colores y como se visualizaba el gráfico, lo deje así y le saque una captura de pantalla para presentarlo.
